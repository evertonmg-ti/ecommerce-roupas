# Arquitetura

## Frontend

- App Router
- Componentes server-first
- Design system simples e reutilizavel
- Area publica separada da area administrativa

## Backend

- API REST versionada
- Modulos independentes por dominio
- Validacao na borda
- Prisma para acesso a dados
- JWT com papeis `ADMIN` e `CUSTOMER`

## Dominios iniciais

- Autenticacao
- Usuarios
- Produtos
- Categorias
- Pedidos
- Dashboard administrativo
